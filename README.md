# deploy1pubmed 
## Local Host:3000 - Instructions
1) Add .env File with API_KEY
```
API_KEY = 'personal-pubmed-api-key'
```
2) Install Dependencies
```
npm install express
npm install axios
npm install papaparse
npm install xml dom 
```
3) Run on local host:3000
```
npm start
```
